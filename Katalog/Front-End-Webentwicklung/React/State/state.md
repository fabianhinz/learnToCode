## State

```jsx
const Counter = () => {
    const [counter, setCounter] = useState(0)

    return (
        <>
            <h1>{counter}</h1>
            <button onClick={() => setCounter(++counter)}>+</button>
            <button onClick={() => setCounter(--counter)}>-</button>
        </>
    )
}
```
